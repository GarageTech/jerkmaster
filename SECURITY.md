# Security Policy

## Reporting A Vulnerability

For software vulnerabilities, open a private security advisory in the GitHub repository:

https://github.com/GarageTech/jerkmaster/security/advisories/new

For urgent safety issues involving heater control, SSR behavior, sensor handling, or emergency-stop behavior, do not publish reproduction steps until the issue has been reviewed.

## Safety Scope

JerkMaster is experimental open-source software for controlling a mains-powered dehydrator. It must be used with independent hardware protection, including a thermal fuse, protective earth, correctly rated over-current protection, an enclosure, and a physical emergency stop.

No software component can guarantee safe operation when wiring, sensors, power supplies, relays, or configuration values are incorrect.
